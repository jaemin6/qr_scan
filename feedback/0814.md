QR코드 인식 및 링크 접속 가능

🔧 변환 논리:
다양한 손상·훼손 데이터 학습 → 깨끗한 QR코드만 테스트

AI 모델 복원 성능 → OpenCV 기본 QR 디코더
QR/바코드/NFC 통합 → QR코드만
자동 조명 조정 → 수동으로 조명 맞춤
인식률 최대화 → 1개 코드 정확히 읽기
